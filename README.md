# Maalaxmi Expense Manager

Maalaxmi is available as a Progressive Web App (PWA) and an Android app (APK). Anyone can install it for free.

## Install Instructions

Option 1 - Install directly from the web (recommended, no download needed). Open https://laxmi.maitpanel.com/ in Chrome (Android) or any modern browser. On Android, tap the menu (three dots) then "Install app" / "Add to Home screen". On Desktop (Chrome/Edge), click the install icon in the address bar. The app installs like a normal app and opens in its own window.

Option 2 - Download the Android APK. Go to the Releases page at https://github.com/maitpanel/maalaxmi-app/releases and download the latest APK file. After downloading on your Android phone, open the APK and tap Install. You may need to allow "Install from unknown sources" in your phone settings.

iPhone (iOS): APK files cannot be installed on iPhone. Use the web version above in Safari, tap Share, then "Add to Home Screen".

## What is this?

Maalaxmi is a web-based expense and income manager for tracking budgets, reports, and financial insights. This repository hosts the installable app versions and the official download links.
